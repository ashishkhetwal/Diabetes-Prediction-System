# Diabetes Prediction System

A beginner Machine Learning project that predicts whether a person is diabetic or not using a Support Vector Machine (SVM) classifier.

This project was built as part of my Machine Learning learning journey and helped me understand the complete workflow of a supervised learning problem.

## Concepts Used

* Data Preprocessing
* Data Standardization
* Train-Test Split
* Support Vector Machine (SVM)
* Model Evaluation

## Results

* Training Accuracy: 78.6%
* Testing Accuracy: 77.2%

## What I Learned

Through this project, I learned how to prepare data for machine learning, train an SVM model using scikit-learn, evaluate its performance, and make predictions on new input data.
